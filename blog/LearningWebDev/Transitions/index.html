Transitions

Learning some Javascript!


My website is PURELY HTML and CSS.. So for my first project outside of my website, I wanted to do something different. How about learn and apply some Javascript!? And that's what I did with my this project that I call Transitions

So basically I wrote a very simple javascript function that changes the color of the background whenever you click on the Click Me button. Cool right?

Javascript is an actual programming language, unlike CSS and HTML - although some would argue that they are as well. Thanks to my Python knowledge, I was able to pick up a ton of the Javascript basics in no time as they, and most programiming languages, share alot of basic features. Such as for-loops, while-loops, variables, lists, strings, integers, floats, conditionals, classes and much more.

Java script does have a couple things that I thought were pretty cool, like it's amazing and time saving shorthands.
For example, a single line arrow function doesn't need HALF of the code for a regular funtion due to its assume return feature. For simple boolean conditionals, you can write what's called a Ternary conditional statement also takes about half of the code a normal conditional would take. (I will edit this in the near future with example code to better illistrate). Another nice feature of Javascript, and most other languages for that matter, is that it isn't white space sensitive like Python is, but nested loops and conditions can get ugly real fast with all of those brackets.
