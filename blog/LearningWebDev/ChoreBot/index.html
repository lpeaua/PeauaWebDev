ChoreBot

March 2nd, 2019

Thank you Codecademy!

**NOTE. This project I followed from Codecademy at the end of the first project of the "Building Interactive JavaScript Websites " course.

I thought this was one of the most useful projects I had done up to that point on Codecademy, as it taught me how to create a simple, fully funtioning and resettable game with sound logic to avoid "alternative" winning methods. It will most definitely help me to create other simple game apps that I will be following this up with.

I also picked up some more HTML and CSS skills. I learned the table HTML tag and how to use the CSS flex display property to format the table tags.

Most importantly, I learned about the DOM, or Domain Object Model, and how to manipulate it using Javascript.
